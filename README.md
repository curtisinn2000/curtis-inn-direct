# Curtis Inn & Suites Direct Booking

Production booking website and admin portal for Curtis Inn & Suites in Hollywood, Florida.

## Stack

- Frontend: React, Vite, TypeScript, Tailwind CSS
- Backend: Node.js/Express in `backend/`
- Hosting: Vercel frontend, Google Cloud Run backend

## Common Commands

```bash
npm run dev
npm run build
npm run lint
npm --prefix backend run build
```

Use `C:\Program Files\Git\bin\bash.exe deploy.sh` from this folder to run the safe deployment flow.
